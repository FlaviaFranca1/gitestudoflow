Pull Request é quando você envia uma sugestão de melhora para o repositório. Quando você quer trazer, pegar puxar algo para seu repositório usando o git, você usa o comando git pull . Você deve adicionar um título, um comentário e , em seguida, pressionar o botão "Create pull request". Neste ponto, os mantenedores do repositório original decidirão se aceitam ou não o seu pull request. Eles podem solicitar que você edite ou revise seu código antes de aceitar o pull request
